# 🌤️ Weather ETL Pipeline — Петропавловск-Камчатский

ETL-пайплайн для сбора данных о погоде с визуализацией.

## 🛠️ Технологии
- Python (pandas, requests, sqlalchemy)
- Apache Airflow — оркестрация
- Docker — контейнеризация
- Flask + Plotly — визуализация
- SQLite — хранение данных

## 📊 Что делает проект
1. Каждый день в 09:00 забирает прогноз погоды из Open-Meteo API
2. Трансформирует данные (ощущаемая температура, переименование колонок)
3. Сохраняет в SQLite базу данных
4. Отдаёт красивые графики через веб-интерфейс

## 🚀 Запуск
```bash
docker compose up -d
python app.py
